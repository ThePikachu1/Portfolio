To add and commit using a single command, do git commit -am 'The commit message' here the -a or -m or -am is called the flag

While performing git actions, if the terminal shows origin/main it refers to the main branch of the remote repository named origin. The origin is a default name for the remote repository that you cloned from or added as a remote. The main branch is the default branch in many repositories, and it is where the primary development work typically happens. So, when you see origin/main, it indicates that you are working with the main branch of the remote repository named origin.
